# 🛒 Online Retail Sales Database (PostgreSQL)

A PostgreSQL project demonstrating intermediate SQL fundamentals using an online retail sales database.

## Topics Covered

- CREATE TABLE
- INSERT INTO
- UPDATE
- DELETE
- DISTINCT
- NULL Handling
- CASE WHEN
- SELECT
- WHERE
- ORDER BY
- LIMIT

## Database

Online Retail Sales Database

## Tables

- customers
- orders

## Technologies

- PostgreSQL
- SQL

## Files

### schema.sql

Creates all database tables.

### data.sql

Populates the database with sample records.

### queries.sql

Contains practical SQL queries demonstrating CRUD operations and SQL fundamentals.

## How to Run

Create a database.

```sql
CREATE DATABASE online_retail;
```

Connect to the database.

Run:

```sql
\i schema.sql
```

Run:

```sql
\i data.sql
```

Run:

```sql
\i queries.sql
```

## Learning Outcomes

This project demonstrates SQL operations commonly used in retail, analytics, and AI/ML data preparation, including data modification, duplicate elimination, null handling, conditional logic, filtering, and sorting.
