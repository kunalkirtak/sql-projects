# 🎓 Student Management Database (PostgreSQL)

A beginner-friendly SQL project demonstrating the fundamentals of PostgreSQL database development.

## Topics Covered

- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- LIMIT

## Database

Student Management System

## Tables

- students

## Technologies

- PostgreSQL
- SQL

## Files

schema.sql

Creates database tables.

data.sql

Inserts sample data.

queries.sql

Contains practice and interview-oriented SQL queries.

## How to Run

Create a database.

```sql
CREATE DATABASE student_management;
```

Connect to it.

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```

## Learning Outcome

This project demonstrates the core SQL concepts required for beginners preparing for Data Analyst, AI/ML, and Software Engineer interviews.
