# 👨‍💼 Employee Management Database (PostgreSQL)

A PostgreSQL SQL project demonstrating SQL Fundamentals through an Employee Management System.

## Topics Covered

- CREATE TABLE
- INSERT
- SELECT
- WHERE
- ORDER BY
- LIMIT
- DISTINCT
- UPDATE
- DELETE
- NULL Handling
- CASE WHEN

## Database

Employee Management

## Tables

- employees

## Technologies

- PostgreSQL
- SQL

## How to Run

Create the database.

```sql
CREATE DATABASE employee_management;
```

Connect to it.

Run:

```sql
\i schema.sql
```

Run:

```sql
\i data.sql
```

Run:

```sql
\i queries.sql
```

## Learning Outcomes

This project demonstrates the SQL fundamentals required for Data Analyst, AI/ML, and Software Engineering roles by managing employee records and performing common business queries.
