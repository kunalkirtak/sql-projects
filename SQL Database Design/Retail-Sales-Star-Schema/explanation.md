# Star Schema

This project demonstrates a Star Schema.

A Star Schema consists of:

- One Fact Table
- Multiple Dimension Tables

The fact table stores measurable business data.

Dimension tables store descriptive information.

Example:

Fact

Sales Amount

Quantity Sold

Dimensions

Customer

Product

Store

Date

This design provides faster analytical queries and is widely used in Data Warehouses, Power BI, Tableau, Microsoft Fabric, Snowflake, Amazon Redshift, BigQuery, and Machine Learning pipelines.
