# 📊 Employee Analytics using SQL Window Functions (PostgreSQL)

A beginner-friendly PostgreSQL project demonstrating the most important SQL Window Functions used in Data Analytics, AI/ML, and Software Engineering interviews.

---

## 📚 Topics Covered

- Window Functions
- OVER()
- PARTITION BY
- ORDER BY
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- NTILE()
- LAG()
- LEAD()
- FIRST_VALUE()
- LAST_VALUE()
- Window Frames
- Running Totals
- Moving Averages
- Cumulative Calculations

---

## 🛠 Technologies

- PostgreSQL
- SQL

---

## 📂 Project Files

### schema.sql

Creates all required tables.

### data.sql

Inserts sample employee data.

### queries.sql

Contains all Window Function examples with comments.

## ▶️ How to Run

### Step 1

Create database

```sql
CREATE DATABASE employee_analytics;
```

### Step 2

Connect to the database.

### Step 3

Run

```sql
\i schema.sql
```

### Step 4

Run

```sql
\i data.sql
```

### Step 5

Run

```sql
\i queries.sql
```
