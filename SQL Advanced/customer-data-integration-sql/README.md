# Customer Data Integration using PostgreSQL

A beginner-friendly SQL project demonstrating SQL Set Operations using PostgreSQL.

## Topics Covered

- UNION
- UNION ALL
- INTERSECT
- EXCEPT

## Database

Customer Data Integration

## Tables

- online_customers
- store_customers

## Technologies

- PostgreSQL
- SQL

## Files

### schema.sql

Creates database tables.

### data.sql

Inserts sample customer records.

### queries.sql

Contains SQL queries demonstrating all SQL Set Operations.

## How to Run

Create database

```sql
CREATE DATABASE customer_data_integration;
```

Connect to database.

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```

## Learning Outcomes

This project demonstrates SQL Set Operations used for combining, comparing and filtering datasets in PostgreSQL.
