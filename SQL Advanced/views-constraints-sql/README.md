# 🎓 University Management Database (PostgreSQL)

A PostgreSQL project demonstrating SQL Constraints, Views, and Materialized Views using a simple University Management Database.

---

## Topics Covered

- PRIMARY KEY
- FOREIGN KEY
- UNIQUE
- NOT NULL
- CHECK
- DEFAULT
- CREATE VIEW
- DROP VIEW
- MATERIALIZED VIEW
