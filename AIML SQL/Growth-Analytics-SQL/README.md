# 📈 Growth Analytics & Recommendation SQL (PostgreSQL)

A PostgreSQL project that demonstrates SQL techniques commonly used in AI, Machine Learning, Product Analytics, and Data Science.

Unlike traditional CRUD projects, this repository focuses on preparing analytical datasets and computing business metrics used in recommendation systems, user behavior analysis, A/B testing, customer analytics, and growth engineering.

---

## Topics Covered

- Funnel Analysis
- A/B Testing Metrics
- Sessionization
- Recommendation Dataset Creation
- Customer Lifetime Value (CLV)

---

## Database Tables

- users
- sessions
- events
- orders
- experiments

---

## Technologies

- PostgreSQL
- SQL

---

## Project Structure

schema.sql

Creates all database tables.

data.sql

Loads sample users, sessions, events, orders, and A/B testing data.

queries.sql

Contains interview-oriented SQL queries for analytics and ML use cases.

---

## How to Run

Create a database.

```sql
CREATE DATABASE growth_analytics;
```

Connect to the database.

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```
