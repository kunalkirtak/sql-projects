# PostgreSQL Advanced Data Types

A beginner-friendly PostgreSQL project demonstrating advanced PostgreSQL data types and operators.

## Topics Covered

- UUID
- JSONB
- Arrays
- JSON Operators
- Array Operators

## Technologies

- PostgreSQL
- SQL

## Project Structure

schema.sql

Creates tables.

data.sql

Inserts sample data.

queries.sql

Contains PostgreSQL feature demonstrations.

## How to Run

Create database

```sql
CREATE DATABASE postgres_datatypes;
```

Connect to database

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```

## Learning Outcomes

- Store semi-structured data using JSONB.
- Store multiple values using Arrays.
- Use PostgreSQL JSON operators.
- Use PostgreSQL Array operators.
- Generate UUID values as primary keys.
