# 📊 Retail Sales Analytics (PostgreSQL)

A PostgreSQL project demonstrating SQL aggregation functions using a retail sales database.

## Topics Covered

- COUNT
- SUM
- AVG
- MAX
- MIN
- COUNT(DISTINCT)
- Conditional Aggregation
- CASE inside SUM

## Database

Retail Sales Analytics

## Tables

- customers
- products
- sales

## Technologies

- PostgreSQL
- SQL

## Files

### schema.sql

Creates all database tables.

### data.sql

Inserts sample data into all tables.

### queries.sql

Contains aggregation queries used for analytics and reporting.

## How to Run

Create a database.

```sql
CREATE DATABASE retail_sales;
```

Connect to the database.

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```

## Learning Outcomes

This project demonstrates SQL aggregation techniques used in reporting, business intelligence, data analytics, and AI/ML data preprocessing.
