# 🎓 University Course Enrollment System (PostgreSQL)

A beginner-to-intermediate SQL project built using PostgreSQL to demonstrate different SQL JOIN operations and database relationships.

---

## 📚 Topics Covered

### SQL Joins

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- SELF JOIN
- CROSS JOIN

### Database Relationships

- One-to-One
- One-to-Many
- Many-to-Many

---

## 🛠 Technologies Used

- PostgreSQL
- SQL

---

## 📂 Project Files

| File | Description |
|------|-------------|
| schema.sql | Creates all database tables |
| data.sql | Inserts sample data |
| queries.sql | SQL queries demonstrating JOINs and relationships |

---

## 🗄 Database Name

```sql
CREATE DATABASE university_enrollment;
```

---

## ▶️ How to Run

Connect to the database.

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```
