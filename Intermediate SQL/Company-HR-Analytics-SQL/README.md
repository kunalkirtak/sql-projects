# 👨‍💼 Company HR Analytics (PostgreSQL)

A PostgreSQL project demonstrating SQL aggregation using **GROUP BY**, **HAVING**, and multiple-column grouping.

---

## 📚 Topics Covered

- GROUP BY
- GROUP BY Multiple Columns
- HAVING
- COUNT
- SUM
- AVG
- MAX
- MIN
- COUNT(DISTINCT)
- Conditional Aggregation
- CASE inside SUM

## 📂 Files

### schema.sql

Creates all database tables.

### data.sql

### queries.sql

Contains SQL queries for HR analytics using GROUP BY and aggregation functions.

---

## ▶️ How to Run

Create a database.

```sql
CREATE DATABASE company_hr;
```

Connect to the database.

Run

```sql
\i schema.sql
```

Run

```sql
\i data.sql
```

Run

```sql
\i queries.sql
```
